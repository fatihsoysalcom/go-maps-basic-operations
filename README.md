# Go Maps Basic Operations

This example demonstrates fundamental operations with maps in Go. It covers how to declare and initialize a map using `make()`, add and access key-value pairs, check for key existence with the 'comma ok' idiom, iterate over map elements, and delete entries.

## Language

`go`

## How to Run

1. Save the code as `main.go`.
2. Run from your terminal: `go run main.go`
